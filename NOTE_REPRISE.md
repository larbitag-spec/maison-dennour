# Note de reprise - Maison d'Ennour

Dernière session : 19/05/2026

## État actuel

- Maquette locale disponible via `ouvrir-maquette.bat` ou `http://127.0.0.1:8080/`.
- Page d'accueil structurée avec header, recherche, menu catalogue, slider, rayons, produits, avis et footer.
- Menu catalogue rapproché du site Maison d'Ennour : dropdown vertical, rubrique active rouge, téléphone à droite.
- Slider principal avec 6 slides :
  - 2 slides Coran existantes.
  - 4 slides "Les Histoires des Prophètes" petit format : blanc, bleu clair, saumon clair, violet clair.
- Les 4 variantes "Les Histoires des Prophètes" sont aussi ajoutées dans la grille "Notre sélection".
- Les boutons "Voir le produit" des slides petit format ouvrent la fiche produit correspondante.
- Les problèmes d'affichage du slider ont été corrigés : hauteur augmentée, contenu compacté, points non chevauchés.
- Mini panier latéral fonctionnel : ajout au panier, compteur, quantité, total.
- Clic sur une carte livre : ouverture d'une fiche produit locale avec image, titre, prix, description, onglets et produits similaires.
- Menu mobile fonctionnel avec ouverture latérale.

## Fichiers principaux

- `index.html` : structure de la page, slider, menu catalogue, fiche produit et panier.
- `styles.css` : design global, responsive, catalogue, slider, fiche produit et panier.
- `script.js` : slider, menu mobile/catalogue, panier, fiche produit, onglets.
- `assets/nouveaux-livres/` : couvertures utilisées dans la maquette.
